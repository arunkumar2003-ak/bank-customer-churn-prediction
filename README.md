# Customer Churn Prediction (Flask + ML)

A Machine Learning web app that predicts whether a customer will churn or not.

---

## 🚀 About Project
This project predicts customer churn using a trained ML model (Logistic Regression) and Flask web app.

---

## 🧠 How it works
- User enters customer details
- Flask sends data to ML model
- Model predicts churn or not churn
- Result shown on web page

---

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- NumPy
- HTML

---

## 📊 Output
- Customer WILL CHURN ❌
- Customer WILL NOT CHURN ✅
